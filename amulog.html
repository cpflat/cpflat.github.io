Amulog project

english / japanese

In amulog project, we have constructed a platform for analyzing log data of large-scale systems such as networks.

  • Log template generation framework for diverse algorithms (amulog)
  • Log data preprocessing tool for header parsing and word segmentation (log2seq)
  • Semantic analysis of log messages (amulog-semantics)
  • Causal analysis of log time-series (logdag)
